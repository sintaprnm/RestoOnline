## About
Program resto online merupakan program sederhana yang memiliki beberapa class yang nantinya akan menerima input dan menghasilkan output. <br/>

Program Java ini dirancang untuk dua jenis user dengan akses yang berbeda, yaitu: <br/>
**1. Admin** akan masuk ke halaman menu Admin dan diberikan akses untuk melihat restoran, menambahkan restoran, dan menghapus restoran. <br/>
**2. Customer** akan masuk ke halaman menu Customer dan diberikan akses untuk melihat restoran, membuat pesanan, dan melihat pesanan. <br/>

Kedua jenis user tersebut diberikan akses untuk kembali ke menu login dan melakukan login ulang. <br/>
